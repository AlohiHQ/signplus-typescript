# Webhook

**Properties**

| Name   | Type         | Required | Description                      |
| :----- | :----------- | :------- | :------------------------------- |
| id     | string       | ❌       | Unique identifier of the webhook |
| event  | WebhookEvent | ❌       | Event of the webhook             |
| target | string       | ❌       | Target URL of the webhook        |
